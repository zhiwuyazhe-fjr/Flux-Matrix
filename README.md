## Flux Matrix（Smart Problem Box）

Flux Matrix 是一款面向数学学习的 AI 题库与训练平台，让你把题目“收进来、讲明白、练到会”。

### 你可以做什么
- **一键导入题目**：支持图片与 PDF，自动识别文字与公式
- **清晰渲染**：题目与解析支持 Markdown + LaTeX 公式展示
- **AI 深度解析**：给出关键概念、思路地图与详细步骤
- **AI 助教追问**：围绕当前题目继续提问，获得更深入的讲解
- **同类题训练**：生成相似题、提交答案、智能批改并保存
- **题库管理**：目录树管理、拖拽排序、收藏与回收站

---

## 使用流程（推荐）

1. **导入题目**  
   上传图片或 PDF，确认识别结果并保存到题库。

2. **阅读与解析**  
   打开题目，查看清晰排版的题目正文与 AI 深度解析。

3. **继续追问**  
   在 AI 助教中询问关键步骤、解题方法或延伸思路。

4. **同类题训练**  
   生成相似题进行练习，系统会给出反馈并可保存为新题。

## 适用人群
- 想高效整理错题、复习高数/线代/概率的人
- 需要清晰公式排版与标准解析的学生与老师
- 希望用 AI 辅助理解题目思路的自学者
